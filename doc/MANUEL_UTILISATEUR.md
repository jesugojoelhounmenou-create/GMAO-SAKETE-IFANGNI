# Manuel utilisateur GMAO

## Connexion
1. Ouvrir http://localhost:5500/login.html
2. Saisir email et mot de passe
3. Cliquer sur "Se connecter"

## Espace Soignant

### Signaler une panne
1. Cliquer sur "Signaler une panne"
2. Sélectionner l'équipement
3. Décrire le problème
4. Cliquer sur "SIGNALER"

### Scanner un QR code
1. Cliquer sur "Scanner QR"
2. Positionner le QR code devant la caméra
3. Voir les informations de l'équipement

### Chatbot
1. Cliquer sur l'icône 🤖
2. Décrire le problème
3. Suivre les instructions
