# Guide du technicien biomédical

## Dashboard
Le dashboard affiche :
- Nombre d'équipements
- Disponibilité globale
- Urgences en attente
- Interventions en cours
- Comptes soignants à valider

## Inventaire
- **Ajouter** : Remplir le formulaire, le QR code est généré automatiquement
- **Modifier** : Cliquer sur "Modifier" sur une carte
- **QR code** : Cliquer sur "QR Code" pour l'afficher
- **Imprimer** : Utiliser le bouton "Imprimer QR"

## Maintenances

### Correctives (Pannes)
1. Voir la liste des urgences
2. Cliquer "Prendre en charge"
3. Réaliser l'intervention
4. Remplir le rapport
5. Cliquer "Terminer"

### Préventives
1. Voir les maintenances planifiées
2. Cocher les vérifications
3. Rédiger le rapport
4. Cliquer "Réaliser"

## Stock
- **Entrée** : Nouvelle pièce reçue
- **Sortie** : Pièce utilisée
- **Alerte** : Stock en dessous du seuil

## Rapports CoDIR
Indicateurs clés :
- Disponibilité globale
- MTBF (fiabilité)
- MTTR (réparabilité)
- Coût de maintenance

## Planning de garde
- Les gardes sont affichées dans le calendrier
- Code couleur : bleu (jour), orange (nuit), vert (astreinte)
